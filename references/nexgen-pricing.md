# NexGen Studio Pricing Reference

Use this to choose proposal pricing. These are internal defaults for draft proposals, not public guarantees. Confirm with Anson before sending a final number.

## Pricing Rule

Do not invent a final price. If Anson has not confirmed pricing, use:

```text
[Anson to confirm final price]
```

## Default Offer Lanes

### Website Refresh

Use when the client already has a site and needs a cleaner, more credible version without much new functionality.

- Draft range: `$750-$1,500`
- Typical scope: homepage refresh, 1-3 core pages, contact form, mobile cleanup, SEO basics, analytics.
- Good fit: small budget, existing content, simple service business.
- Watchouts: if the current site is too broken, quote as Custom Business Website instead.

### Custom Business Website

Use for a new website or full rebuild for a small business that needs a professional public presence.

- Draft range: `$1,500-$3,500`
- Typical scope: homepage, about/services page, contact page, gallery/work page if needed, forms, analytics, SEO basics, deployment, DNS support.
- Good fit: local service companies, contractors, owner-operators.
- Watchouts: content delays can stretch timeline. Ask for photos, services, service area, and contact info early.

### Website + Operations Workflow

Use when the website should reduce manual work, not just display pages.

- Draft range: `$3,500-$8,000+`
- Typical scope: custom business website plus quote/request workflow, admin view, document flow, status tracking, scheduling, CRM/SMS/email integration planning.
- Good fit: businesses losing time to repeated texts, forms, spreadsheets, missed leads, paperwork, or manual follow-up.
- Watchouts: scope needs a short discovery call before final pricing.

### Custom Software / Portal

Use when the main deliverable is an authenticated app, dashboard, portal, or internal workflow.

- Draft range: custom quote only.
- Typical scope: process map, authenticated app, database, roles, admin workflows, reporting, deployment.
- Good fit: client has a repeatable operational process that can be software-supported.
- Watchouts: never quote casually from rough notes.

## Proposal Language

Use this when pricing is not final:

> Investment: [Anson to confirm final price]

Use this when a range is appropriate:

> Based on the current scope, this would typically fall in the `$X-$Y` range. I would confirm the final number after the short scope call.

Use this when fixed:

> Investment: `$X`, with final payment due before ownership transfer.

## Add-ons

- Project gallery: quote after number of photos/projects is known.
- Service-area pages: price per page or bundle after service areas are confirmed.
- Google Business Profile cleanup: small fixed add-on if in scope.
- Monthly maintenance: optional retainer, confirm with Anson.
- Estimate request workflow: quote as Website + Operations Workflow.
