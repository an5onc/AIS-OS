# NexGen Studio Offer Notes

Use this when drafting proposals, outreach, and scope recommendations.

## Positioning

NexGen Studio builds websites, internal tools, and custom software for small businesses that need more than a generic template.

The strongest fit is a local service business, contractor, owner-operator, or small team that needs one or more of these:

- A professional mobile-first website.
- Clear lead capture and contact flows.
- SEO basics and city/service pages.
- Admin workflows, document workflows, dashboards, portals, or forms.
- Long-term technical ownership without being locked into a low-flexibility website builder.

## Offer Lanes

### Website Refresh

Use when the client already has a site but it looks outdated, lacks clear calls to action, or needs a cleaner mobile experience.

Typical scope:

- Homepage refresh.
- Core service pages.
- Contact form and clickable phone/email.
- Mobile layout improvements.
- Basic SEO cleanup.
- Analytics setup.

Pricing note: confirm with Anson before quoting. Use as a smaller option when budget is the main constraint.

### Custom Business Website

Use for a new site or full rebuild where the client needs a polished public presence and stronger lead capture.

Typical scope:

- Homepage.
- About page.
- Services pages.
- Gallery/work page if relevant.
- Contact page and lead form.
- Basic local SEO structure.
- Deployment, DNS support, and handoff.

Pricing note: this should usually be the default website proposal lane.

### Website + Operations Workflow

Use when the business also needs the site to reduce manual admin work.

Typical scope:

- Everything in Custom Business Website.
- Lead intake workflow.
- Quote/request form.
- Admin dashboard or simple backend.
- Document, invoice, scheduling, or status workflow if needed.
- Integration planning for email, SMS, payments, CRM, or database.

Pricing note: this is the strongest NexGen differentiator. Use when the client says they are tired of manual follow-up, spreadsheets, missed leads, repeated texts, paperwork, or disconnected systems.

### Custom Software / Portal

Use when the client primarily needs a workflow app, dashboard, portal, or internal tool, with the website as secondary.

Typical scope:

- Discovery and process map.
- Authenticated app or admin portal.
- Database-backed workflows.
- Role-based views if needed.
- Reporting or export features.
- Deployment and maintenance plan.

Pricing note: do not quote this casually. Confirm scope with Anson first.

## Competitive Tradeoff Language

Use language like this:

> A WordPress or template-based site can be a good fit for a business that only needs basic pages and simple edits. Where NexGen Studio is different is that I can build the website and the business workflow behind it, so the site can turn into a system instead of stopping at a brochure.

Avoid:

- Calling competitors bad.
- Claiming custom code is always better.
- Overpromising SEO rankings.
- Quoting final pricing before Anson confirms scope.
