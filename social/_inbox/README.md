# Daily input notes (optional)

Drop a short note here any day you want the posts tailored to something specific. Name the file by date and brand, e.g.:

- `2026-06-06-kiefer-built.md`
- `2026-06-06-interlockgo.md`

Inside, write anything relevant for that day, for example:
- "We just finished a kitchen remodel in Timnath — photo attached: kitchen-timnath.jpg"
- "Push the elevator service this week."
- "InterlockGo: answer the question about rolling retests."

If there is no note for a brand on a given day, the generator uses the normal content-pillar rotation. Notes always override the rotation.
