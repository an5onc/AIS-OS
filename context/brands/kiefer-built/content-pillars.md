# Content pillars: Kiefer Built Contracting

Rotate through these so posts stay varied. Each day, pick the next pillar in order (loop back to the top after the last one). If the day's input note specifies a topic or photo, that overrides the rotation.

1. **Craftsmanship & finish detail** — close-up appreciation of quality work (trim, tile, cabinetry, stonework).
2. **Project showcase** — a completed home, remodel, or commercial space (ONLY with a real photo + permission; if none available, fall back to pillar 1 or 6).
3. **Process & trust** — how Kiefer Built communicates, coordinates trades, and keeps builds organized.
4. **Service spotlight** — highlight one offering (kitchen remodels, additions, basements, elevators, commercial, EPS homes).
5. **Northern Colorado / local** — tie to the area, seasons, communities served (Windsor, Fort Collins, Loveland, Greeley, Timnath).
6. **Team & company values** — 25 years, 200+ projects, careers/hiring, the people behind the work.

## Seasonal awareness
Reference the current real season/month when natural (e.g. spring build season, winter interior remodels). Do not reference holidays unless the input note mentions one.
