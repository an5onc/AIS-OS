# Image strategy: Kiefer Built Contracting

**Default: use REAL project photos. Do NOT generate fake AI homes.**

A custom home builder lives on authenticity. AI-generated houses look generic and can misrepresent actual work, which damages trust.

## Where to get real photos (in priority order)
1. That day's input note in `social/_inbox/` (the human may attach/point to a fresh jobsite photo).
2. A local folder of real Kiefer Built photos if one exists in the repo (check `brand-assets/kiefer-built/` — create nothing, just look).
3. Real photos hosted on the websites. Known image paths you may reference by URL in the post plan:
   - https://www.builtbykiefer.com/images/project-3/exterior-twilight-front.jpg (custom exterior, twilight)
   - https://www.builtbykiefer.com/images/project-3/interior-great-room-wide.jpg (open living space)
   - https://www.builtbykiefer.com/images/project-1/kitchen-8.jpg (kitchen remodel)
   - https://www.builtbykiefer.com/images/project-2/exterior-front-facade.jpg (mountain build exterior)
   - https://www.builtbykiefer.com/images/project-3/interior-master-suite.jpg (primary suite)
   - https://www.builtbykiefer.com/images/project-3/interior-primary-bath.jpg (bathroom)
   - https://www.builtbykiefer.com/images/project-2/exterior-wide-property.jpg (acreage/site)
   - https://www.builtbykiefer.com/images/project-3/kitchen-island-front.jpg (finish detail / island)

## In the post plan, specify the image as ONE of:
- **REAL PHOTO:** a specific URL or filename from the sources above, with a one-line reason it fits the caption.
- **NEEDS PHOTO:** if no suitable real photo exists, write "NEEDS PHOTO — [describe the exact shot Anson should take or pull]" so the human supplies it. Do NOT substitute an AI image.

## Never
- Never generate AI images of homes/interiors for this brand.
- Never use a photo that doesn't match what the caption claims.
