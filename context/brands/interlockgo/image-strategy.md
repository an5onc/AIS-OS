# Image strategy: InterlockGo NOCO

**AI-generated graphics and clean stock-style images are ACCEPTABLE for this brand** (unlike KieferBuilt). There are no "real project" authenticity concerns — the goal is clear, calm, professional visuals.

## Preferred image types
- Clean photo or render of a vehicle dashboard / steering wheel area (device discreetly present, NOT a focal "gotcha").
- Simple branded text graphics for educational/FAQ posts (e.g. "3 steps: Call → Install → Drive").
- Calm Northern Colorado road/driving imagery.
- Real shop/install photos if the human supplies them via the input note (best option when available).

## In the post plan, specify the image as ONE of:
- **REAL PHOTO:** a filename/URL the human supplied (preferred when available).
- **GENERATE:** a complete, ready-to-paste image-generation prompt describing the exact image, including style ("clean, professional, calm, no text unless specified"), aspect ratio (1:1 for Instagram, 1.91:1 or 1:1 for Facebook), and what to avoid ("no shaming imagery, no police, no handcuffs, no alcohol").
- **TEXT GRAPHIC:** describe a simple text/graphic layout the human can make in Canva or Meta tools.

## Never
- Never use imagery implying shame, arrest, jail, handcuffs, police, or alcohol.
- Never imply a legal outcome.
